# ppd-openmp

## Options de compilation

> Pour compiler les programmes, utiliser l'option -fopenmp. Voici la commande à taper :

```
gcc -o <nom_binaire> <fichier.c> -fopenmp
```

> Pour exécuter :

```
./<nom_binaire>
```
